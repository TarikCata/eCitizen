E-Citizen

This project is only a prototype of the actual project and it is developed only for presentation purposes of the idea.

For the development of this project we used Angular and a lot of fun.

The base idea of this project is as the name already says e-country. It's a project to make our lives a lot easier.

Everything you need from country and what requires you to wait in line for one, two or even three hours to get something done, which could be done in 1 minute, now it's only a few clicks of job.

Also, there are many other things we took care of.
