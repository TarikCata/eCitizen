More features we had on our mind...

Connection between countries

This project could be connection between countries so visas and similar things would be easier to get. That means that we could implement this application for all countries and all countries would have different interfaces and features depending on country law.

Reports about money flow

In our country we are facing corruption and many similar things so with this app we could provide feature to show all citizens money flow and where money is spend.

"Extraordinary" voting

This feature represents model where citizens would be able to vote if they are not satisfied, and if those votes are for example above 50% of all citizens, application would run new elections.
